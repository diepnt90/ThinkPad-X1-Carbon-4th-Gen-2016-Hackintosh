# macOS on ThinkPad X1 Carbon (4th Gen 2016)

本教程可使得小黑吃上macOS Catalina (10.15.4)。

## 引导环境
使用[OpenCore](https://github.com/acidanthera/OpenCorePkg/releases)版本0.5.8，所有kext均为当前最新版本。
